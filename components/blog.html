Q-1 what is difference between let const and bar in JavaScript?

In JavaScript, the keywords let, var, and const are used to declare variables. Variables declared using var are scoped to the function in which they are created. let and const are block scoped, meaning they are only accessible within the immediate set.

Q-2 What is the difference between find, filter, methods?

The find() method is used to find all descendant elements of the selected element. The filter method is used to filter all elements.
